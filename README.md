# 批量文件重命名脚本

> 只对文件进行重命名
> 切换到脚本目录

    查看帮助
    node rename --help
    
    用法
    node rename -src src -dist dist -n test- -i 100
    
| 参数 | 说明 |
|---|
| -src | 处理文件源目录，默认：src |
| -dist | 生成文件存放目录，默认：dist |
| -n | 文件名称前缀，默认：image | 
| -i | 文件索引，默认：0 |
